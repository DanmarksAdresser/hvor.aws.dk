Om Hvor er jeg?

Hvor er jeg? giver dig - ud fra din position - oplysninger, hvor i Danmark du er.

Formålet med applikationen er give et eksempel på, hvad Danmarks Adressers Web API - DAWA kan bruges til. Kildeteksten til applikationen finder du på GitHub.

Applikationen er udviklet af Styrelsen for Dataforsyning og Effektivisering

Du kan holde dig orienteret om DAWA og tilknyttede aktiviteter i AWS-gruppen på Digitaliser.dk samt på Twitter
